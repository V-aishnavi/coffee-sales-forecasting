# Coffee Sales Forecasting Using Machine Learning

## Project Overview

This project analyzes coffee shop sales data and develops machine learning models to forecast daily revenue. 
The analysis combines exploratory data analysis, feature engineering, predictive modeling, and interactive dashboards.

---

## Objectives

• Analyze historical coffee sales patterns  
• Identify high-performing products  
• Forecast daily revenue using machine learning  
• Build an interactive dashboard for business insights  

---

## Dataset

The dataset contains transaction-level coffee sales including:

- Date
- Coffee type
- Revenue

The data was aggregated into daily revenue for time-series forecasting.

---

## Exploratory Data Analysis

Key insights explored:

- Daily revenue trends
- Monthly sales performance
- Product revenue distribution
- Weekday vs weekend sales patterns

---

## Model Comparison

Models tested:

- Linear Regression
- Random Forest
- Gradient Boosting
- XGBoost

Random Forest achieved the best performance for this dataset.
